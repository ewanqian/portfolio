## Working Rule

后续重新开放首页索引时，建议只保留满足下面条件的稿子:

1. 能明确对应到真实项目、真实工具或真实活动
2. 不把规划写成已完成事实
3. 不把个人或小团队实践夸张成行业级平台
4. 读者读完后，能知道“你实际做了什么”，而不是只感到宏大
