Artistic Overview

创作总览

钱誉文的实践围绕时间、图像、空间与感知之间的连续关系展开。早期工作首先集中在声音如何进入图像内部，并在节奏、频段与段落结构中形成视觉组织；随后，图像逐渐从单一屏幕关系中展开，进入身体、环境与空间尺度之中；近年的重点则进一步延伸到数字环境、扫描生态、双目观看与感知迁移系统。网页、舞台、环幕、数字展览、档案与接口研究，在这里构成的并非彼此独立的媒介分支，而是一条持续推进的创作网络。

Temporal Structure / 时间结构

节拍、频段、能量与段落变化被逐步纳入视觉内部，成为作品的组织材料。

Spatial Generation / 空间生成

图像开始与身体、环境、尺度和屏幕结构共同生成一种空间性的经验。

Perceptual Migration / 感知迁移

作品在不同界面和空间条件中的出现方式，构成了实践本身的重要部分。

Creative Timeline

创作时间线

时间作为材料 / Temporal Structure

这条线始于对声音与图像关系的长期处理。节拍、频段、能量与段落变化被逐步纳入视觉内部，成为作品的组织材料。TIMER 系列集中呈现了这一阶段的核心问题：时间如何被塑造成一种可见的结构，并在图像内部建立起持续的秩序。

Read More / 详细说明 Related Works / 相关作品

图像进入空间 / Spatial Generation

随着实践推进，工作的重心逐渐转向图像与空间之间的关系。Drop Flow 相关作品标志着这一阶段的展开：图像开始与身体、环境、尺度和屏幕结构共同生成一种空间性的经验。观看不再停留于画面表层，而进入一种由流动、密度与尺度共同构成的场域。

Read More / 详细说明 Related Works / 相关作品

环境成为作品 / Perceptual Environments

扫描、点云、数字采样与观察系统，使实践进一步进入数字环境本身。在这一阶段，关注点延伸到环境如何被组织、数字空间中的生命感如何成立，以及未完成状态、碎片化结构与共生关系如何在技术媒介中持续显现。

迁移作为延续 / Perceptual Migration

近年的工作逐步收束到“感知迁移”这一核心框架。网页、舞台、环幕、双目、空间计算、档案与研究文本，被纳入同一条持续发展的结构线索之中。作品在不同界面和空间条件中的出现方式，构成了实践本身的重要部分；迁移由此成为作品得以延续、展开与被重新阅读的方法。
