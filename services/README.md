## What I Actually Help With / 我实际会帮你解决什么

很多合作方不一定一开始就知道自己需要买哪种服务。  
更常见的情况是：

- 想做一个短片，但不知道风格怎么定
- 想做演出视觉，但不知道预算怎么拆
- 想做双目 / XR 内容，但不知道平台要求
- 想整理一个数字展览，但不知道从作品到页面怎么组织
- 想优化工程、流程或内容结构，但不知道找谁诊断

所以我通常不只是"执行"，也会同时协助处理：

- 脚本与方向梳理
- 视觉风格测试
- 数字环境与空间内容开发
- 内容版本整理
- 展示逻辑与页面组织
- workflow 优化与项目结构诊断
