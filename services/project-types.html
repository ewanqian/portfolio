Project Types / 服务分类说明

按「从低门槛到高价值」的顺序，把我能接的项目类型整理成几条清晰的入口。

01. AI 视觉短视频与传播内容

适合品牌传播、企业形象、活动预热与社交媒体内容开发。我可以负责：

  • 脚本方向梳理
  • 视觉风格测试
  • AI 图像与视频生成
  • 基础镜头与节奏组织
  • 短内容的出片测试

参考起步区间：¥1000 起（15秒短视频）

02. 舞台视觉 / 演出影像

适合演唱会、音乐节、发布会、live set 与表演现场。我更适合：

  • 有一定艺术追求、不只是要"能用就行"的现场
  • 注重空间氛围、段落节奏与观看体验的项目
  • 愿意把视觉内容也作为整体创作一部分的合作

参考起步区间：¥3000 起（单条内容），¥8000–30000+（开场视觉）

03. 空间影像 / 沉浸式内容

适用于投影空间、多面屏、环幕、展厅内容与沉浸式展示。重点是：

  • 不只是单一画面，而是内容如何进到空间里
  • 考虑多屏适配、投影逻辑、环境风格
  • 包括内容版本与展示入口的组织

参考起步区间：¥5000 起

04. 双目视频 / Apple Vision Pro / XR

适合双目影像、空间观看与沉浸式数字环境方向。我会把：

  • 双目 / stereo 内容
  • 数字环境
  • 空间观看方式

一起考虑。

建议路径：先从测试版、样片、proof of concept 开始。

参考起步区间：¥5000 起（样片/测试版）

05. 数字展览 / 展示系统 / 线上专展

适用于艺术家线上专展、数字展览、作品整理与展示系统搭建。这里不只是搭页面，更包括：

  • 展品分类
  • 文案整理
  • 观看路径
  • 展示入口
  • 归档与后续可持续阅读结构

适合已经有作品，但还没有清晰展示系统的合作方。

参考起步区间：¥8000 起

06. 工程调试 / 工作流咨询 / 内容整理

如果你已经有工程、内容或项目，但结构混乱、交付不稳或流程不清，这类服务会比重做整个项目更高效。我可以：

  • 工程结构检查
  • 问题定位与建议
  • 工作流优化
  • 内容结构诊断
  • 视觉系统优化

参考起步区间：¥800 起（工程诊断轻服务）

07. 课程 / 培训 / 工作坊

适合院校、机构、企业培训和创意团队内部训练。内容方向包括：

  • VR / XR 基础
  • AI 视觉工作流
  • 数字展览与展示系统
  • 项目组织与创作流程优化

参考起步区间：¥4000–6000 起（半天工作坊）
