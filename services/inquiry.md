# Inquiry / 合作咨询说明

这一页用于帮助合作方在联系前快速整理需求，也用于后续网页表单与内部 skills 的字段设计参考。

如果你已经准备好要聊一个项目，建议第一次联系时尽量提供以下信息。这样我可以更快判断项目类型、预算区间、合作方式，以及是否适合先从咨询、测试样片或完整制作开始。

---

## What to Prepare / 建议准备的信息

### 1. 项目类型
请尽量说明你要做的是哪一类项目，例如：

- AI 视觉短视频 / 品牌传播内容
- 舞台视觉 / Opening Visual / 演出背景
- 空间影像 / 投影 / 沉浸式内容
- 双目视频 / Apple Vision Pro / XR 内容
- 数字展览 / 线上专展 / 展示系统
- 工程调试 / workflow 咨询 / 内容整理
- 课程 / 培训 / 工作坊 / 讲座
- 其他定制项目

### 2. 项目用途
请说明内容最终会用在哪里，例如：

- 抖音 / 小红书 / 社交媒体传播
- 品牌活动 / 发布会
- 演唱会 / live set / 音乐节
- 展厅 / 投影空间 / 沉浸式展示
- Apple Vision Pro / XR 设备展示
- 学校课程 / 企业培训 / 工作坊
- 艺术家个人展示 / 数字展览

### 3. 项目规模
尽量说明以下信息中的一部分：

- 片长或内容时长
- 是单条还是系列内容
- 是否多版本输出
- 是否多屏 / 多平台
- 是否需要现场配合
- 是否已有脚本 / 素材 / 工程 / 参考方向

### 4. 时间节点
请尽量说明：

- 希望什么时候开始
- 最晚什么时候交付
- 是否有固定上线 / 演出 / 展出日期
- 是否属于赶工项目

### 5. 预算范围
如果方便，建议直接告诉我预算范围，例如：

- ¥1000–3000
- ¥3000–8000
- ¥8000–20000
- ¥20000+
- 暂时不确定，希望先咨询

### 6. 当前状态
你也可以说明目前已经做到哪一步：

- 只有一个想法
- 已有脚本 / brief
- 已有素材
- 已有工程
- 已找过其他人做了一半
- 需要从零开始整理

## Recommended First Step / 建议的合作起点

如果项目还比较模糊，我通常会建议从以下几种方式先开始：

### 1. 咨询
适合先把需求讲清楚、判断方向和预算范围。

### 2. 测试样片 / Proof of Concept
适合双目 / XR、舞台视觉、空间影像等方向，先验证风格、结构和技术路线。

### 3. 提案包 / 视觉方向测试
适合品牌项目、活动发布、展览和前期方向确认。

### 4. 完整制作
适合目标已经清楚、预算和交付方式较明确的项目。

### 5. 工程诊断 / workflow 咨询
适合已有工程、内容或流程，但结构混乱、需要外部判断和优化建议的情况。
