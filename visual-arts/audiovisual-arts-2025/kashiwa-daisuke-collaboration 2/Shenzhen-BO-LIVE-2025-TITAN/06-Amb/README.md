# 06 Amb

## 作品信息

- **创作时间**：2025年10月
- **格式**：动态影像 / 现场视觉
- **所属项目**：深圳 BO LIVE「机械光合：TITAN的全息声林」

## 作品说明

高氛围 ambient 视觉方向。远离激进的音频分析逻辑，追求漂浮、治愈、几乎超然的感觉。

## 视觉关键词

- ambient
- 氛围
- 漂浮
- 治愈
- 超然
