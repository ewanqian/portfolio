## 关于

与日本音乐人 Kashiwa Daisuke 合作的视觉艺术系列。该系列目前包括深圳 BO LIVE 专场与 CAN Festival 现场项目，涵盖现场视觉、音画结构、空间生成与氛围段落设计。

**核心关键词：**
- 视听表演 (Audiovisual Performance)
- 现场视觉 (Live Visuals)
- 时间结构 (Temporal Structure)
- 空间生成 (Spatial Generation)
