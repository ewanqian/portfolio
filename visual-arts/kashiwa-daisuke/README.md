## 关于

与日本音乐人 Kashiwa Daisuke 合作的视觉艺术系列。该系列目前包括深圳 BO LIVE 专场与 CAN Festival 现场项目，涵盖现场视觉、音画结构、空间生成与氛围段落设计。

**核心关键词：**
- 视听表演 (Audiovisual Performance)
- 现场视觉 (Live Visuals)
- 时间结构 (Temporal Structure)
- 空间生成 (Spatial Generation)

---

## Collaboration Overview / 合作概览

该系列整理的是我与日本音乐人 KASHIWA Daisuke 及合作音乐人 Yuki Murata 的现场视觉合作档案。目前已公开的项目包括深圳 BO LIVE 专场《机械光合：TITAN 的全息声林》与舟山 CAN Festival 2025 pt.2。

这条合作线主要围绕以下方向展开：
- 现场视听结构
- 音乐驱动的视觉段落
- 空间生成与氛围控制
- 舞台语境中的视觉协作

## Credits Structure / 合作结构

该系列项目通常由以下层级共同构成：
- 音乐演出：KASHIWA Daisuke、Yuki Murata 及相关现场音乐人
- 视觉创作：Ewan Qian
- 现场制作系统：策划、统筹、技术、灯光、声音、舞台与场地团队共同完成

每个项目页将进一步列出对应场次的详细演职人员与执行信息。
