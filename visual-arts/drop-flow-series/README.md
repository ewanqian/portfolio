Drop Flow 系列围绕空间生成、沉浸式图像、身体表演与图像离开屏幕展开，是我近年关于空间叙事与感知迁移的重要作品线之一。它关心的不只是画面本身，而是图像如何与空间、身体、观看位置和光线关系一起工作。

## Why This Series Matters

这是我作品线中最能体现“空间生成”与“进入机制”的一条主线。它并不把图像当成平面内容，而是尝试让观众感受到自己正在进入一个被组织过的状态之中。

如果你想先理解我如何处理沉浸式空间、现场结构与图像外溢，建议从这个系列开始。

## Core Focus

- 空间生成 / Spatial Generation
- 沉浸式图像 / Immersive Imaging
- 身体与空间关系 / Body-Space Relation
- 图像离开屏幕 / Image Leaving the Screen
- 感知迁移 / Perceptual Migration
