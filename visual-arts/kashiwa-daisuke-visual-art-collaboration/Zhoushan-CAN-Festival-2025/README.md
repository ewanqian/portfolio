## 项目说明

这场表演延伸了早期合作中开发的视听语言，同时回应了现场乐队设置、舞台灯光与音乐节规模的表演条件。相比以往基于投影的环境，这次版本进一步推向现场扩展、节律控制与舞台响应的视觉结构。
