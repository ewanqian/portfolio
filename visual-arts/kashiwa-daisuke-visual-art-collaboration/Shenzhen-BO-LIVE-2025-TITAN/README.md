## 项目说明

这场特别视听表演围绕《TITAN》的氛围与声音世界展开。视觉语言聚焦于雾气、光线、白空间深度、环境过渡、沉浸错觉与感知运动。部分段落采用强烈的音频驱动结构，部分段落则追求更柔和、漂浮、治愈与沉思的状态。
